 What is java?       

Java is a programming language used for creating software that makes
it compatible with many different operating
environments. It is a two-stage programming language, meaning that it is a compiled language and an interpreted one. However, it differs from most other
compiled languages because it does not compile directly to an
executable file.Java is an object-oriented, class-based, concurrent, secured and general-purpose computer-programming language. It is a widely used robust technology.
